Video Mania

Aplicacion para la creación de sitios de publicación de concursos de videos.

* Ruby version: ruby 2.1.5p273 (2014-11-13 revision 48405) [i386-mingw32]

* System dependencies: Instalación de ffmpeg

* Database creation: Utilización de MySQL


* Services (job queues, cache servers, search engines, etc.)
Antes de iniciar la ejecución de la aplicación correr: rake jobs:work
